# agri-sat-scan

Comandos para limpiar el proyecto de Android:

```bash
rm -rf android/.gradle
flutter clean
rm -rf ~/.gradle/caches/
./gradlew clean
./gradlew build
```

Comando para limpiar el proyecto de Flutter:

```bash
flutter clean
```

Comando para reconstruir los paquetes de Flutter:

```bash
flutter pub get
```

Comando para iniciar el proyecto de Flutter:

```bash
flutter run
```
